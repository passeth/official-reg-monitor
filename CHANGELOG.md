# Changelog

## 0.1.0

- Added installable `official-reg-monitor` CLI.
- Added source registry for KR, EU, US, JP, CN, ASEAN, and RU/EAEU.
- Added raw source snapshot capture with SHA-256 change detection.
- Added same-origin asset capture for SPA-style official sources.
- Added SQLite schema for raw snapshots, assets, parser runs, and normalized regulatory rows.
- Added parser scaffold with US eCFR XML part detection and JP PDF provenance handling.
- Added macOS launchd schedule template.
- Added smoke tests and GitHub Actions CI.
